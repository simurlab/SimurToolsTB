f_carga_DOT – Cargar y procesar archivos .csv de sensores Xsens Dot

La función f_carga_DOT permite cargar datos desde un archivo .csv generado por sensores Xsens Dot, procesar la columna de tiempo (SampleTimeFine) y opcionalmente guardar la tabla procesada como archivo .mat.

Syntax

tabla = f_carga_DOT();
Error using f_carga_DOT (line 35)
❌ No se encontró ningún archivo .csv en la ruta: /Users/juan/GitHub/SimurToolsTB/docs_src
tabla = f_carga_DOT('folder_path', 'ruta/a/archivos');
tabla = f_carga_DOT('save', 'y');
 

Description

Parámetro
Tipo
Descripción
Valor por defecto
folder_path
string
Ruta a la carpeta con archivos .csv
Carpeta actual
save
char
'y' para guardar la tabla como dot_YYMMDD.mat; 'n' para no guardar
'n'

Output

  • tabla – Objeto de tipo table con las columnas del CSV, incluyendo una columna Time normalizada en segundos.

Examples

% Usar valores por defecto (carpeta actual, sin guardar)
tabla = f_carga_DOT();
 
% Graficar un ejemplo si existe una señal de aceleración
if any(strcmp(tabla.Properties.VariableNames, 'FreeAcc_X'))
plot(tabla.Time, tabla.FreeAcc_X);
xlabel('Tiempo (s)');
ylabel('Aceleración libre X');
title('Señal de aceleración en el eje X');
end
 

Notes

  • La función utiliza el primer archivo .csv encontrado en la carpeta.
  • Asegúrate de que el archivo contenga la columna SampleTimeFine.
