importfile_xsens

It reads a text file generated with an xsens IMU, and all signals are renamed and stored. It is a file automatically generated by the matlab data importer, with small modifications.

Syntax

[Counter,Temperature,Acc_X,Acc_Y,Acc_Z,Gyr_X,Gyr_Y,Gyr_Z,Mag_X,Mag_Y,Mag_Z] = importfile_xsens(filename);

Description

importfile_xsens(filename) necesita el nombre del archivo de texto del xsens.

Examples

% Cargamos los datos de un experimento de calibracion ya conocido:
filename = '00500686-xsens.txt';
[Counter,Temperature,Acc_X,Acc_Y,Acc_Z,Gyr_X,Gyr_Y,Gyr_Z,Mag_X,Mag_Y,Mag_Z] = importfile_xsens(filename);
