cadencia

Calcula la cadencia de carrera a partir de los eventos de foot-strike.

Syntax

[cadencia_valores, cadencia_media] = cadencia(IC, freq);

Description

cadencia calcula la cadencia de carrera (en pasos por minuto) utilizando los eventos de foot-strike contenidos en IC y la frecuencia de muestreo freq. Devuelve un vector cadencia_valores con la cadencia instantánea entre foot-strikes consecutivos y el valor medio cadencia_media correspondiente a toda la señal.
El parámetro de entrada IC debe ser un vector o celda con los índices de los eventos de foot-strike (talonazo). freq contiene la frecuencia de muestreo en Hz.

Examples

Supongamos que disponemos de los índices de foot-strike detectados en una prueba de carrera:
% Simulamos los índices de foot-strike (cada 0.6 segundos)
freq = 100; % Hz
IC = num2cell(round(0.6*freq:0.6*freq:10*freq));
Calculamos la cadencia
cad = cadencia(IC, freq);
Representamos el resultado gráficamente.
disp(['Cadencia media: ', num2str(cad, '%.2f'), ' pasos/min']);
Cadencia media: 106.67 pasos/min
Notas
- La cadencia se calcula como el número de pasos por minuto:
cadencia = (60 * freq) ./ diff(IC);
- Es recomendable eliminar los primeros y últimos eventos si se producen irregularidades.
- Esta función forma parte del módulo de análisis de dinámica de carrera de la SIMUR Toolbox.
